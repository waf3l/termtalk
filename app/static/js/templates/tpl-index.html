Command list with short description
- nick [nick] - get and set nick name
- create_room -d [description] -p [password for private room] - if password is not none then the room is private else room is public
- join_room -n [name of room] -p [password for private room ] - join room
- leave_room - leave room
- list_rooms -c -l - count or list all rooms
If you join room, then you only need to typ message and press enter without any command prefix
